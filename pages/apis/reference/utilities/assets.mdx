# Assets

## **[GET]** /v1/:network/assets

Get all assets of the current chain

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: cosmos
```

<TryAPI
  method="GET"
  host="https://apis.mintscan.io"
  url="/v1/:network/assets"
  useBearerAuthorization
/>
