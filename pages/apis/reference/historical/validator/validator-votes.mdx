# Validator Votes

## **[GET]** /v1/:network/validators/:validatorAddress/votes

Get a list of votes of a validator

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: cosmos
# The address of validator
# (string, required)
validatorAddress: cosmosvaloper1clpqr4nrk4khgkxj78fcwwh6dl3uw4epsluffn
```

```yaml filename="Queries"
# Whether to remove duplicates for the same proposal
# (boolean, optional, default: false)
distinct: true
```
