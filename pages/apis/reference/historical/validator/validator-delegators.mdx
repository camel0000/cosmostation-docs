# Validator Delegators

## **[GET]** /v1/:network/validators/:validatorAddress/delegators

Get a list of richlist of a symbol

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: cosmos
# The address of account
# (string, required)
validatorAddress: cosmosvaloper1clpqr4nrk4khgkxj78fcwwh6dl3uw4epsluffn
```

```yaml filename="Queries"
# Request per page
# (number, optional, default: 20)
take: 10
# Value of skip amount for pagination
# (number, optional, default: 0)
from: 20
```
