# Asset Holders

## **[GET]** /v1/:network/holders

Get a list of token holders by denom

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: injective
```

```yaml filename="Queries"
# The token denom
# (string, required)
denom: inj

# Optional keyword to filter holder address
# (string, optional)
keyword: inj1fl48vsnmsdzcv85q5d2q4z5ajdha8yu3lj7tt0

# Number of holders to return
# (integer, optional, default: 20, max: 1000)
limit: 10

# The number of records to skip for pagination (Use with `limit` to paginate through the results.)
# (integer, optional, default: 0)
offset: 20
```
